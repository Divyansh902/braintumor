 (MNIST)



This project uses a Deep Learning model to recognize handwritten digits (0–9) using the MNIST dataset.

---

## ⚙️ Technologies

* Python 3.10
* TensorFlow / Keras
* Matplotlib

---

## 🧠 Model

* Flatten Layer (28×28 → 784)
* Dense Layer (128 neurons, ReLU)
* Output Layer (10 neurons, Softmax)

---

## ▶️ How to Run

```bash
python3.10 -m pip install tensorflow==2.15 matplotlib
python3.10 mnist_model.py
```

---

## 📊 Output

* Accuracy ≈ 95%
* Predicted vs Actual digit
* Sample image display
