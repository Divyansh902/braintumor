# CIFAR-10 CNN Classification


This project uses a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset into 10 categories.

## 📊 Dataset
- 60,000 images (32x32 RGB)
- 10 classes (airplane, car, bird, cat, etc.)

## 🧠 Model
- Conv → Pool → Conv → Pool → Conv → Flatten → Dense → Output

## ⚙️ Technologies
- Python
- TensorFlow / Keras
- Matplotlib

## ▶️ Run
pip install tensorflow matplotlib  
python cifar10_cnn.py

## 📈 Result
Accuracy: ~65–70%
